AI-enhanced shoutouts generate a personalized message about a streamer using Google Gemini, posted alongside the standard `!so` / `!shoutout` message.

## How It Works

1. A moderator uses `!so <username>` in chat.
2. The standard shoutout message is always sent first (name, link, last game).
3. If AI shoutouts are enabled, the bot fetches the target streamer's profile data from Twitch Helix:
   - Channel info (description, game, title)
   - Follower count
   - Top clips
4. This data is sent to Google Gemini with a prompt requesting a brief, fun shoutout message.
5. The AI-generated message is posted to chat as a second message.

If the AI call fails for any reason, the standard shoutout still works — AI enhancement is non-blocking.

## Requirements

### Environment Variables

| Variable | Required | Description |
|----------|----------|-------------|
| `GEMINI_API_KEY` | Yes | Google Gemini API key |
| `AI_SHOUTOUT_ENABLED` | Yes | Set to `"true"` to enable globally |

Both must be set for AI shoutouts to be available. See [Environment Variables](/docs/getting-started/environment-variables) for setup.

### Per-Channel Toggle

Even with the global environment variables set, AI shoutouts must also be enabled per-channel:

- **Dashboard:** Go to Settings → Features → toggle "AI-Enhanced Shoutouts"
- The toggle is only visible when `GEMINI_API_KEY` and `AI_SHOUTOUT_ENABLED` are configured

## Technical Details

- **Model:** Gemini 2.0 Flash (`gemini-2.0-flash`)
- **Timeout:** 5 seconds per request
- **Cache:** 30-second per-user cache to prevent spam
- **Max output:** 150 tokens (~1-2 sentences)
- **Temperature:** 0.8 for creative but consistent output
