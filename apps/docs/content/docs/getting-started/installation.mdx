## Prerequisites

- [Node.js](https://nodejs.org/) v20 or later
- [Bun](https://bun.sh/) v1.x or later
- [Docker](https://www.docker.com/) (for PostgreSQL and Redis)
- A [Twitch application](/docs/getting-started/create-twitch-app) with client ID and secret
- A [Discord application](/docs/getting-started/create-discord-app) with a bot token

## Clone and Install

```bash
git clone https://github.com/mrdemonwolf/community-bot.git
cd community-bot
bun install
```

## Start Infrastructure

```bash
docker compose up -d postgres redis
```

This starts PostgreSQL on port 5432 and Redis on port 6379.

## Configure Environment

Copy the example env files for each app:

```bash
cp apps/web/.env.example apps/web/.env
cp apps/discord/.env.example apps/discord/.env
cp apps/twitch/.env.example apps/twitch/.env
```

Each `.env.example` file includes comments explaining every variable. See [Environment Variables](/docs/getting-started/environment-variables) for the full reference.

## Initialize Database

```bash
bun db:push        # Push schema to database
```

## Start Development

```bash
bun dev            # Starts all apps in dev mode
```

Or start individual apps:

```bash
bun dev --filter discord-bot
bun dev --filter twitch-bot
bun dev --filter docs
```

## What to Expect

On first startup, the web app logs a one-time setup URL to the console. Visit the URL to complete the [setup wizard](/docs/getting-started/setup-wizard) — this configures the broadcaster, authorizes the bot's Twitch account, and promotes you to admin.

If something doesn't work, check the [Troubleshooting](/docs/getting-started/troubleshooting) page.
